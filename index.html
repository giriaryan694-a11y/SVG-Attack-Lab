01 — Select Attack Technique
XSS
Inline Script — onload XSS
Injects JavaScript via SVG onload event handler. Fires when SVG is rendered inline in HTML.
Critical
XSS
Script Tag XSS
Embeds a <script> block directly inside SVG. Works when SVG is served as text/html or embedded inline.
Critical
EXFIL
Image-based Data Exfiltration
Uses <image href> to beacon data to an attacker server by encoding it in query parameters.
High
DOM XSS
foreignObject DOM Injection
Embeds HTML namespace inside SVG via <foreignObject>. Bypasses SVG-only sanitizers.
Critical
XXE
XML External Entity (XXE)
DOCTYPE with external entity reads local files server-side. Requires vulnerable XML parser.
Critical+
COOKIE THEFT
Cookie Exfiltration via Fetch
Uses fetch() inside SVG to transmit document.cookie to a controlled endpoint.
Critical+
